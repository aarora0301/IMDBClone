# IMDBClone
Create IMDB clone using Node and Mongo</br>

## To run project:</br>
1.Clone Repo</br>
2.run 'npm install'</br>
3.run 'npm start'</br>

### Endpoints created:

**User API**:</br>
1.POST: UserRegistration-->http://localhost:8282/api/users/ </br>
sample payload:</br>
{</br>
    "user":{</br>
    	"email":"aarshi@gmail.com",</br>
        "password":"ioio"</br>
    }</br>
}</br>

2.POST: LoginUser-->http://localhost:8282/api/users/login</br>
sample payload:</br>
{</br>
    "user":{</br>
    	"email":"aarshi@gmail.com",</br>
        "password":"ioio"</br>
    }</br>
}</br>
